SelChanged (Item/Value)
 
Function Name c:<CONTROL-NAME>_OnSelChanged
Syntax   (defun c:<CONTROL-NAME>_OnSelChanged (ItemIndex [as Long] Value [as String] /)
)
Return Type Ignored
Applies To Combo Box, Image Combo Box, List Box, Option List

This event is triggered after the current selection has changed. ItemIndex is the index of the newly selected item, and Value is the item text.
