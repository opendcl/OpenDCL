GetBlockSize
 
Function Name dcl_GetBlockSize
Syntax   (dcl_GetBlockSize BlockName [as String])
Return Type List of (Width [as Long] Height [as Long])

This function returns the extents of the specified block. If the block is not defined in the current drawing, the function returns NIL.
