Show OK
 
Friendly Name Show OK
API Name ShowOK
Type Boolean
Restrictions  
Get Function   (dcl_Control_GetShowOK <CONTROL>)
Set Function   (dcl_Control_SetShowOK <CONTROL> NewValue [as Boolean])
Applies To File Explorer

If this property is set during initialization, the Open/Save button is shown; otherwise it is hidden.
