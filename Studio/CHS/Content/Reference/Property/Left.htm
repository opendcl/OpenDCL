Left
 
Friendly Name Left
API Name Left
Type Long
Restrictions  
Get Function   (dcl_Control_GetLeft <CONTROL>)
Set Function   (dcl_Control_SetLeft <CONTROL> NewValue [as Long])
Applies To All Controls

This property determines the X coordinate (in pixels) of the upper left corner of the control. The coordinate is relative to the upper left corner of the control's parent window (higher values are farther right on the display). The value of this property is synchronized with the Left From Right and Width properties. The Use Left From Right property controls the relationship of these properties.
