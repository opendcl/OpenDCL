Return As Tab
 
Friendly Name Return As Tab
API Name ReturnAsTab
Type Boolean
Restrictions  
Get Function   (dcl_Control_GetReturnAsTab <CONTROL>)
Set Function   (dcl_Control_SetReturnAsTab <CONTROL> NewValue [as Boolean])
Applies To Combo Box, Image Combo Box, Text Box

If this property is set and the user presses [Enter] while the control has the keyboard focus, the [Enter] is interpreted as though it was [Tab].
