Dropdown Height
 
Friendly Name Dropdown Height
API Name DropdownHeight
Type Long
Restrictions Must be greater than or equal to zero
Get Function   (dcl_Control_GetDropdownHeight <CONTROL>)
Set Function   (dcl_Control_SetDropdownHeight <CONTROL> NewValue [as Long])
Applies To Combo Box, Image Combo Box

This property determines the minimum height (in pixels) of the control's dropdown list box. Changing this property at runtime has no effect after the control has been created.
