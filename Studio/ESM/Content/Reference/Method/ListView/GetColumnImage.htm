ListView_GetColumnImage
 
Function Name dcl_ListView_GetColumnImage
Syntax   (dcl_ListView_GetColumnImage <CONTROL> Column [as Long])
Return Type Integer
Applies To List View

This method returns the column header image index from the control's image list for the specified column. If the list is not in report view mode, this function returns NIL.
