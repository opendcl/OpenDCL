Begriffserläuterung
 

Dialog

Ein Dialog ist eine Vorlage (Klasse), von der zur Laufzeit eine Instanz für eine Dialogbox innerhalb von AutoCAD abgeleitet wird. Dialoge werden im OpenDCL Studio interaktiv erstellt und verändert, indem neue Steuerelemente dem Dialog hinzugefügt und den Steuerelementen und der Dialogbox selbst Eigenschaften zugewiesen werden.

Steuerelement

Ein Steuerelement ist ein einzelnes Element eines Dialoges. Normalerweise ist ein OpenDCL-Steuerelement analog zu einem Windows-Steuerelement (wie Textfelder, Schaltflächen usw.). Einige Steuerelemente sind jedoch etwas komplexer und beinhalten mehrere Windows-Steuerelemente, die nur gemeinsam funktionieren. Bitte beachten Sie, dass in OpenDCL Dialoge auch wie Steuerelemente behandelt werden.
