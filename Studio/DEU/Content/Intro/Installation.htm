Installation
 

OpenDCL Installation

OpenDCL Studio lässt sich sehr leicht installieren. Laden Sie die gewünschte Sprachversion von der OpenDCL Studio Download Webseite herunter und führen Sie die MSI-Datei aus. Dialoge können Sie im OpenDCL Studio auch ohne AutoCAD-Lizenz erstellen und verändern. Um die gesamte Entwicklungsarbeit auf einem einzigen Computer zu leisten und beispielsweise den Quellcode und das Verhalten der Dialoge und Steuerelemente testen zu können, ist es jedoch empfehlenswert, das AutoCAD während der Entwicklung installiert ist. Das OpenDCL Studio enthält die OpenDCL Laufzeitumgebung zum Laden und Anzeigen der Dialoge innerhalb von AutoCAD.

OpenDCL sollte vorzugsweise lokal installiert werden. Während der Installation wird die OpenDCL-Laufzeitumgebung in allen unterstützten Versionen von AutoCAD eingerichtet, die auf dem Arbeitsplatz installiert sind.

OpenDCL Beispiele

OpenDCL Studio liefert viele verschiedene Beispiele mit aus, die die unterschiedlichen Dialoge und Steuerelemente einschließlich ihrer Verwendung demonstrieren. Um die Beispiele zu starten, starten Sie AutoCAD und laden Sie die Datei "Samples\_MasterDemo.lsp", die im länderspezifischen Unterordner Ihrer OpenDCL-Installation gespeichert ist. Die Datei kann durch Drag&Drop vom Windows Explorer in den Zeichenbereich der aktuellen Zeichnung geladen werden, um OpenDCL in Aktion treten zu lassen. Bitte beachten Sie, dass Sie keine zusätzlichen Supportpfade ergänzen müssen, um die OpenDCL-Beispiele zu laden und anzuwenden.
