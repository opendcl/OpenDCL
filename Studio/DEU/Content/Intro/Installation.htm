Installation
 

OpenDCL Studio lässt sich sehr leicht installieren. Laden Sie die gewünschte Sprachversion von der OpenDCL Studio Download Webseite herunter und führen Sie die MSI-Datei aus. Dialoge können Sie im OpenDCL Studio auch ohne AutoCAD-Lizenz erstellen und verändern. Um die gesamte Entwicklungsarbeit auf einem einzigen Computer zu leisten und beispielsweise den Quellcode und das Verhalten der Dialoge und Steuerlemente testen zu können, ist es jedoch empfehlenswert, das AutoCAD während der Entwicklung installiert ist. Das OpenDCL Studio enthält die OpenDCL Laufzeitumgebung zum Laden und Anzeigen der Dialoge innerhalb von AutoCAD.

OpenDCL sollte vorzugsweise lokal installiert werden. Während der Installation wird die OpenDCL-Laufzeitumgebung in allen unterstützten Versionen von AutoCAD eingerichtet, die auf dem Arbeitsplatz installiert sind.
