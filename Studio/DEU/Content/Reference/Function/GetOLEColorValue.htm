GetOLEColorValue
 
Function Name dcl_GetOLEColorValue
Syntax   (dcl_GetOLEColorValue AcadColor [as Color])
Return Type OLEColor

This function converts an AutoCAD color or system color into an OLEColor for use with ActiveX properties and methods.
