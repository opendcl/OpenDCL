Row Height
 
Lokaler Name Row Height
API Name RowHeight
Datentyp Long
Gültigkeit Must be greater than or equal to -1
Get Funktion   (dcl_Control_GetRowHeight <CONTROL>)
Set Funktion   (dcl_Control_SetRowHeight <CONTROL> NewValue [as Long])
Anwendbar auf DWG List, Grid, Option List

This property sets the height of a row in the list. A value of 0 uses the default height for the control; a value of -1 uses a varying height based on the size of the content.
