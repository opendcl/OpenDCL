Style (Kombinationsfeld)
 
Lokaler Name Style
API Name ComboBoxStyle
Datentyp Integer
Gültigkeit Range is 0 - 14
Get Funktion   (dcl_Control_GetComboBoxStyle <CONTROL>)
Set Funktion   (dcl_Control_SetComboBoxStyle <CONTROL> NewValue [als Integer])
Anwendbar auf Kombinationsfeld, Kombinationsfeld mit Bildern

This property determines the style of combo box. Some styles are not available for the image combo box.

Value Meaning Combo Box Image Combo Box
0 Standard combo list box with edit control X X
1 Simple combo list box without edit control X X
2 Dropdown combo box with edit control X X
3 AutoCAD arrow styles combo box X  
4 AutoCAD colors combo box X  
5 AutoCAD lineweights combo box X  
6 AutoCAD plot style names combo box X  
7 AutoCAD plot style tables combo box X  
8 Dropdown font combo box X  
9 Simple font combo list box X  
10 AutoCAD plotters combo box X  
11 AutoCAD plotter paper sizes combo box X  
12 Folder selection combo box X  
13 AutoCAD layers combo box X  
14 AutoCAD linetypes combo box X
