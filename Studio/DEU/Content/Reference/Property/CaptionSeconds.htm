Caption (Seconds)
 
Lokaler Name Caption (Sekunden)
API Name CaptionSeconds
Datentyp String
Gültigkeit  
Get Funktion   (dcl_Control_GetCaptionSeconds <CONTROL>)
Set Funktion   (dcl_Control_SetCaptionSeconds <CONTROL> NewValue [als String])
Anwendbar auf Verlaufsbalken

The custom unit of time label to use for zero, two, or more seconds when displaying the time remaining. If this property is empty, the default localized label (e.g. "seconds") will be used.
