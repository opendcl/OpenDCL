Legend Type
 
Lokaler Name Legend Type
API Name LegendType
Datentyp Integer
Gültigkeit Range is 0 - 2
Get Funktion   (dcl_Control_GetLegendType <CONTROL>)
Set Funktion   (dcl_Control_SetLegendType <CONTROL> NewValue [als Integer])
Anwendbar auf Verlaufsbalken

This property determines the style of legend to display on the progress bar. Note that this property is ignored and no legend is displayed if Use Visual Style is true.

Value Description
0 None
1 Time remaining
2 Percent completed
