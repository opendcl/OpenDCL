Funktionsname c:<CONTROL-NAME>_OnDragOverFromControl
Syntax   (defun c:<CONTROL-NAME>_OnDragOverFromControl (strProjectName [als String] strFormName [als String] strControlName [als String] uDropPoint [variiert] /)
)
Rückgabewert Bitflag
Ausgelöst durch Blockliste, Block, DWG-Liste, DWG-Vorschau, Bildschaltfläche, Schraffur, Bezeichnung, Listenfeld, Erweiterte Liste, Bild, Dia, Textfeld, Textschaltfläche, Baumstruktur

This event is triggered as the user passes the cursor over the control during a drag and drop operation from another OpenDCL control (or from this control). The return value from this function is used to determine which type of action will occur if the drop is made at the current point. The ProjectName, FormName, and ControlName arguments identify the control from which the drag operation was initiated. The DropPoint argument is a 2DPoint in most cases, with the following exceptions.

Zielsteuerelement Datentyp von uDropPoint
Baumstruktur mit Schlüsseln für die Einträge String (Schlüssel des Elements, auf dem die Maustaste losgelassen wurde)
Baumstruktur ohne Schlüsseln für die Einträge Handle (Handle des Zieleintrags)
Alle anderen Steuerelemente 2DPunkt (Mauskoordinaten in Abhängigkeit des Steuerelements)

Return Value Behavior
NIL Default
0 None
1 Copy
2 Move
4 Link
