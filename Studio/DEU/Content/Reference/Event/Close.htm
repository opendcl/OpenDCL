Close
 
Funktionsname c:<CONTROL-NAME>_OnClose
Syntax   (defun c:<CONTROL-NAME>_OnClose (UpperLeftX [as Long] UpperLeftY [as Long] /)
)
Rückgabewert Ignored
Ausgelöst durch All Forms

This event is triggered immediately before the dialog is closed. UpperLeftX, UpperLeftY is the coordinate of the upper left corner of the dialog.
