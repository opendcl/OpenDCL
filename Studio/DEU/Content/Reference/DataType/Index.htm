Datentypen
 

OpenDCL supports the following data types used for function arguments and function return types.

  • 2DPunkt
  • 3DPunkt
  • ActiveX Objekt
  • Bitflag
  • Boolean
  • Farbe
  • Control
  • Datum
  • Float
  • Handle
  • Image List
  • Integer
  • Long
  • OLEColor
  • PictureID
  • String
