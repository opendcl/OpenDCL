OpenDCL Beispiele

Zweck dieser Beispiele ist die Demostration verschiedener Funktionen von OpenDCL.
Die Datei @AllSamples.lsp zeigt eine Werkzeugleiste mit Schaltflächen zum Start
jedes Beispiels. Schieben Sie die @AllSamples.lsp einfach in den Zeichenbereich
von AutoCAD.

Hinweis: Sie müssen das Verzeichnis mit den Beispielen nicht als Supportpfad hinzufügen.
