Horizontal Scroll Bar
 
Friendly Name Horizontal Scroll Bar
API Name HScrollBar
Type Boolean
Restrictions  
Get Function   (dcl_Control_GetHScrollBar <CONTROL>)
Set Function   (dcl_Control_SetHScrollBar <CONTROL> NewValue [as Boolean])
Applies To Text Box

If this property is set, a horizontal scroll bar is displayed when necessary.
