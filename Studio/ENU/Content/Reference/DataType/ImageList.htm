Image List
 

Image lists are Windows system objects for holding an ordered list of icons, and they are represented by handles in Windows. OpenDCL encodes image list handles as entity names when returning them to AutoLISP. These values are not really entity names, but they can be passed to OpenDCL functions that expect an image list, and OpenDCL will interpret them correctly as image list handles.
