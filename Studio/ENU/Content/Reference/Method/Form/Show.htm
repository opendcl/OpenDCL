Form_Show
 
Function Name dcl_Form_Show
Syntax   (dcl_Form_Show <CONTROL> {NewWidth [as Long]} {NewHeight [as Long]})
Return Type Boolean (T if successful; otherwise NIL)
Applies To Control Bar, File Dialog, Modal Dialog, Modeless Dialog, Palette

This method centers the dialog on the display. If the optional NewWidth and NewHeight arguments are included, the form is resized before it is centered.
