AxObject_Put
 
Function Name dcl_AxObject_Put
Syntax   (dcl_AxObject_Put <AXOBJECT> PropertyName [as String] NewValue [as Varies])
Return Type Boolean (T if successful; otherwise NIL)
Applies To ActiveX Object

This method sets the value of a COM property exposed by an ActiveX object. A NIL return value indicates failure.
