Large Change
 
Friendly Name Large Change
API Name LargeChange
Type Long
Restrictions  
Get Function   (dcl_Control_GetLargeChange <CONTROL>)
Set Function   (dcl_Control_SetLargeChange <CONTROL> NewValue [as Long])
Applies To Scroll Bar, Slider

This value sets the size of the change in value when a page up or page down input is received by the control.
