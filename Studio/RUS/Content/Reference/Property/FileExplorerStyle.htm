Style (File Explorer)
 
Friendly Name Style
API Name FileExplorerStyle
Type Integer
Restrictions Range is 0 - 1
Get Function   (dcl_Control_GetFileExplorerStyle <CONTROL>)
Set Function   (dcl_Control_SetFileExplorerStyle <CONTROL> NewValue [as Integer])
Applies To File Explorer

This property determines the style of file explorer control.

Value Meaning
0 Open file dialog
1 Save file dialog
