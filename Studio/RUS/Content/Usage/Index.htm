Using OpenDCL
 

Using OpenDCL requires familiarity with OpenDCL Studio to design user interface components and AutoLISP to display and control them inside AutoCAD at runtime. This documentation assumes familiarity with AutoLISP and a basic understanding of creating and using AutoLISP code in AutoCAD.
