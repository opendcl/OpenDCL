ButtonClicked (Row/Column)
 
Function Name c:<CONTROL-NAME>_OnButtonClicked
Syntax   (defun c:<CONTROL-NAME>_OnButtonClicked (Row [as Long] Column [as Long] /)
)
Return Type Ignored
Applies To Grid

This event is triggered when the user clicks on a button edit control in the cell at the specified row and column.
