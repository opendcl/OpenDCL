Insert Or Xref
 
Friendly Name Insert Or Xref
API Name InsertOrXref
Type Integer
Restrictions Range is 0 - 1
Get Function   (dcl_Control_GetInsertOrXref <CONTROL>)
Set Function   (dcl_Control_SetInsertOrXref <CONTROL> NewValue [as Integer])
Applies To DWG List

This property controls what happens by default during a drag and drop operation from this control onto AutoCAD.

Value Description
0 Insert as block
1 Attach as external reference
