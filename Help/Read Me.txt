Help Compilation Instructions:

Extract all files and folders from this archive maintaining directory structure.

Install Vizacc Helpmaker http://www.vizacc.com/ and open the OpenDCL.sh6 project file. (Helpmaker is not necessary for further editing as source .htm files can be edited independently.

Ensure all images within the 'images' folder are included in the project search path - use the Baggage tab.

Compile the project in Helpmaker. Helpmaker will recreate all .htm files to reflect any edits to topics.

Copy and paste the 3 .htm files in the 'special files' folder into the _tmphhp folder following the HelpMaker compilation, overwriting existing files. This is required as the audtutorial.htm and tool toolbar.htm files are standalone files that have been edited outside of Helpmaker due to limitations of the software.  

Open the opendcl.hhp file in the '_tmphhp' directory using Microsoft HTML Help Workshop (included in the installation of Helpmaker). Edit the font for the main window by clicking Project Options (top left button on project tab) to Verdana.

Save and recompile the project from Help Workshop.
